# Ruby on Rails Expert Protocol

## Core Mandates
1. **Core Framework**: Default to Ruby 4.0.1 and Rails 8.1.2 for all core application logic and AI pair-programming services. 
2. **Performance Engine**: Enable ZJIT for Just-In-Time compilation. Utilize Ractor for concurrent, isolated background task execution.
3. **Frontend Architecture**: Enforce Hotwire (Turbo and Stimulus) for dynamic page updates without full reloads. Do not integrate heavy JavaScript SPA frameworks unless explicitly required by the Twin-Graph frontend architecture.
4. **Security & Auth**: Implement Rails 8 built-in authentication mechanisms to manage sessions. Enforce robust password hashing and data encryption methods provided by the core framework.
