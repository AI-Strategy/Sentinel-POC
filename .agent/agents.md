## 6. Boundaries & Safeguards
* **Restricted Files**: `.env`, `credentials.json`, `*.pem`
* **Restricted Operations**: 
    * **No SQLite**: Use Postgres.
    * **No "Port Hopping"**: Kill zombie processes instead of opening new ports.
    * **No Regex**: Use Gemini Flash for complex parsing.
* **Code Style**:
    * **Strict Typing** always.
    * **Async I/O** for all Database/Network calls.
    * **Document** every new module in `/docs` and update `README.md`.

## 6. Global Directives (Summary)
1. **Software Currency**: Trust no training data (18mo+ old). Never downgrade software/models without explicit permission. Verify versions via Web Search.
2. **Data Persistence**: Enterprise-grade DBs only (Postgres/Neo4j).
3. **Network**: Static Port Allocation (3000/8000/5432/6379).
4. **Security**: Zero-Leak Policy. Secrets in `.env`.
5. **Restraint**: Optimize for clarity. Delete unused code.
6. **FinOps**: Clean up unused resources immediately.
